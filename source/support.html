Frequently asked questions

Things are broken for me, where can I find help?

Please don't hesitate to contact us by email at support@opener-project.org. However, if you have problems with a specific component, you can also go to the Github repository of the component and create an issue in the issue tracker.


Can I use all of this for free?

Most of it, yes. Unless noted otherwise OpeNER software uses an Apache V2 license.
